# u8array
![Travis (.com)](https://img.shields.io/travis/com/nothingrandom/u8array)
> Create Uint8Array from string

## Install

``` bash
$ yarn add u8array
$ npm install u8array
```

## Usage (require in JS)
``` js
const u8 = require('u8array');

const uint8array = u8('test string');
```
